The code in this project was used to obtain the results of the paper 

Itzhakov, A., Codish, M.: Complete symmetry breaking constraints for the class
  of uniquely hamiltonian graphs. Constraints 27(1), 8--28 (Apr 2022).

https://link.springer.com/article/10.1007/s10601-021-09323-8


The code depends on the following libraries:
Choco 4 constraint solver https://github.com/chocoteam/choco-solver
Apache Commons CLI https://commons.apache.org/proper/commons-cli/
